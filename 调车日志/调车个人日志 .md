    ### 完整调车流程
1. 开环确认机械：轮子不卡、重心居中、舵机中值正确、图像清晰无畸变~~惯性导航？？**~~
2. 调扫线+二值化阈值：确保白底黑线稳定扫出边界，大津法下限防黑暗误判、上限防反光
3. 纯P跑闭环：从低速(0.5m/s)开始，调Kp让车沿中线走完一圈
4. 加前馈：标定前馈表，让弯道入角更主动，减少PID负担
5. 加Kd抑制震荡：重点调S弯和急弯，让车丝滑过弯
6. 加Ki消除跑偏：直道侧偏、出弯回不到中线时加
7. 提速迭代：逐渐提高速度，每次提速后微调P/D/I和前馈表
8. 元素专项：逐个攻克圆环→三叉→坡道→十字，每个元素配独立参数集
9. 保护机制：OFFLine>55激活停车保护，防冲飞出赛道    
## 6.11
- 由于硬件原因，即使程序写的电机旋转方向都是顺时针或逆时针同一方向旋转，但右电机安装后旋转方向反，因此需要程序中取反
```cpp
bool motor_init()
{
    // 设置电机方向为默认状态（根据硬件调整）
    right_dir.set_level(1);      // 右电机方向初始化
    left_dir.set_level(1);       // 左电机方向初始化
    
    // 设置PWM占空比为0，确保电机停止
    right_motor.set_duty(0);     // 右电机PWM清零
    left_motor.set_duty(0);      // 左电机PWM清零
    
    return true;
}
```

```cpp
  // -------------------- 步骤4：方向控制+PWM输出 --------------------
    // ⚠️注意：硬件接线导致left/right反接，所以代码中需要交换
    // right_pwm控制左电机，left_pwm控制右电机
    
    // 控制左电机（使用right_pwm）
    if (right_pwm <= 0) {  // PWM为负，需要反转
        left_dir.set_level(1);              // 设置方向为反转
        left_motor.set_duty(-right_pwm);    // PWM取绝对值输出
    } else {               // PWM为正，正转
        left_dir.set_level(0);              // 设置方向为正转
        left_motor.set_duty(right_pwm);     // PWM直接输出
    }
    
    // 控制右电机（使用left_pwm）
    if (left_pwm <= 0) {   // PWM为负，需要反转
        right_dir.set_level(0);             // 设置方向为反转
        right_motor.set_duty(-left_pwm);    // PWM取绝对值输出
    } else {               // PWM为正，正转
        right_dir.set_level(1);             // 设置方向为正转
        right_motor.set_duty(left_pwm);     // PWM直接输出
    }
    ```
