使用vofa+远程连接了小车，但有时候vofa+连接不上小车，需要重新交叉编译，调参cmds脚本版本落后，具体使用方法还在探索.....

臃肿的tabby仅用来串口连接小车核心版用来执行`ifconfig`命令来查看小车的ip地址

使用ubuntu自带终端来ssh连接小车
`ssh root@000.00.00.000`(板子的ip地址)

使用终端以及vscode对板子的linux文件进行操作
1. 安装工具
`sudo apt install sshfs`
2. 新建文件夹
`mkdir -p ~/桌面/xiaoche`
3. 挂载文件夹
`sshfs root@192.168.79.125:/home/root ~/桌面/xiaoche`
4. 强制卸载挂载
`fusermount -uz ~/桌面/xiaoche`
5. 使用vscode编辑或者ubuntu文件管理器你哈

    你好
